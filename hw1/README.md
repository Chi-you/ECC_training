# BPSK modulation
## How to run
* Compile, execute and plot two results
```
$make all
```
* Run the modulation without hamming code
```
$make hw1-1
```
* Run the modulation with hamming code
```
$make hw1-2
```
* To remove all the file
```
$make clean
```
